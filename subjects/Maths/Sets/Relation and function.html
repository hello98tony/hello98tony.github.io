Introduction to Relations & Functions

Relations and Functions form the foundation of higher mathematics and are crucial for JEE preparation. They describe how elements from different sets are related to each other.

Relation: A subset of the Cartesian product A × B that defines a relationship between elements of set A and set B.

Function: A special type of relation where each element of set A is related to exactly one element in set B.

Basic Terminology

  • Domain: Set of all first elements in the ordered pairs (input values)
  • Range: Set of all second elements in the ordered pairs (output values)
  • Codomain: The set to which the function maps the domain
  • Image: The value assigned to a particular element of the domain
  • Pre-image: The element(s) of the domain that map to a particular element in the range

JEE Focus: Questions often involve identifying types of relations/functions, composition of functions, inverse functions, and graphical interpretations.

JEE Problem Example:

Let A = {1, 2, 3} and B = {4, 5}. Find the number of relations from A to B.

Solution: Number of elements in A × B = 3 × 2 = 6
Number of subsets of A × B = 2⁶ = 64
∴ Number of relations from A to B = 64

Relations

A relation R from set A to set B is a subset of A × B. When (a, b) ∈ R, we say "a is related to b".

Types of Relations

Relation Type Definition Example
Empty Relation R = ∅ ⊂ A × A No elements are related
Universal Relation R = A × A All elements are related
Identity Relation Iₐ = {(a, a) | a ∈ A} Each element related to itself
Reflexive (a, a) ∈ R ∀ a ∈ A ≥ relation on ℝ
Symmetric (a, b) ∈ R ⇒ (b, a) ∈ R "Is sibling of" relation
Transitive (a, b), (b, c) ∈ R ⇒ (a, c) ∈ R "Is ancestor of" relation
Equivalence Reflexive, Symmetric, and Transitive Congruence modulo n
Number of relations from A to B = 2n(A)×n(B)
JEE Problem Example:

Check whether the relation R in ℝ defined as R = {(a, b) : a ≤ b²} is reflexive, symmetric, or transitive.

Solution:
Not reflexive: (1,1) ∉ R since 1 ≤ 1² is false when 1 = 1²
Not symmetric: (1,4) ∈ R but (4,1) ∉ R
Not transitive: (-2,0) ∈ R and (0,-1) ∈ R but (-2,-1) ∉ R

Functions

A function f: A → B is a special relation where each element of A appears exactly once as the first component in the ordered pairs.

Function Characteristics

  • Domain: Complete set of possible input values
  • Codomain: Set of possible output values
  • Range: Actual set of output values achieved
  • One-to-One (Injective): f(x₁) = f(x₂) ⇒ x₁ = x₂
  • Onto (Surjective): Range = Codomain
  • Bijective: Both injective and surjective
Vertical Line Test: A curve is a function if no vertical line intersects it more than once

JEE Focus: Function problems often involve domain/range determination, checking injectivity/surjectivity, and piecewise functions.

JEE Problem Example:

Find the domain of the function f(x) = √(x - 1) + √(3 - x).

Solution:
x - 1 ≥ 0 ⇒ x ≥ 1
3 - x ≥ 0 ⇒ x ≤ 3
∴ Domain = [1, 3]

Types of Functions

Functions can be classified based on their mapping properties and algebraic forms.

Standard Function Types

Function Type Definition Example
Polynomial f(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₀ f(x) = x² + 3x + 2
Rational Ratio of two polynomials P(x)/Q(x) f(x) = (x+1)/(x-2)
Exponential f(x) = aˣ, a > 0 f(x) = 2ˣ
Logarithmic f(x) = logₐx, a > 0, a ≠ 1 f(x) = ln x
Modulus f(x) = |x| f(x) = |x - 3|
Signum sgn(x) = -1, 0, or 1 sgn(-5) = -1
Greatest Integer f(x) = [x] [3.7] = 3

Special Functions

Even Function: f(-x) = f(x) (symmetric about y-axis)
Odd Function: f(-x) = -f(x) (symmetric about origin)
Periodic Function: f(x + T) = f(x) for some period T
JEE Problem Example:

Check if f(x) = (aˣ + a⁻ˣ)/(aˣ - a⁻ˣ) is even, odd, or neither.

Solution:
f(-x) = (a⁻ˣ + aˣ)/(a⁻ˣ - aˣ) = (aˣ + a⁻ˣ)/-(aˣ - a⁻ˣ) = -f(x)
∴ f(x) is odd

Composition of Functions

The composition of functions f and g is defined as (g ∘ f)(x) = g(f(x)).

Properties of Composition

  • Associative: h ∘ (g ∘ f) = (h ∘ g) ∘ f
  • Not commutative: f ∘ g ≠ g ∘ f in general
  • Identity: f ∘ I = I ∘ f = f (where I is identity function)

Important Note: For g ∘ f to be defined, range of f must be subset of domain of g.

If f: A → B and g: B → C are both injective, then g ∘ f is injective
If f: A → B and g: B → C are both surjective, then g ∘ f is surjective
JEE Problem Example:

If f(x) = x² and g(x) = 2x + 1, find (f ∘ g)(x) and (g ∘ f)(x).

Solution:
(f ∘ g)(x) = f(g(x)) = f(2x + 1) = (2x + 1)² = 4x² + 4x + 1
(g ∘ f)(x) = g(f(x)) = g(x²) = 2x² + 1

Inverse Functions

A function f⁻¹ is the inverse of f if f⁻¹(f(x)) = x for all x in domain of f and f(f⁻¹(y)) = y for all y in range of f.

Conditions for Inverse

  • A function must be bijective (one-to-one and onto) to have an inverse
  • The graph of f⁻¹ is the reflection of f's graph about y = x
  • Domain of f becomes range of f⁻¹ and vice versa
To find f⁻¹:
1. Set y = f(x)
2. Solve for x in terms of y
3. Replace y with x to get f⁻¹(x)

JEE Focus: Inverse trigonometric functions are particularly important for JEE problems.

JEE Problem Example:

Find the inverse of f: ℝ → ℝ defined by f(x) = (3x + 2)/(x - 1).

Solution:
Let y = (3x + 2)/(x - 1)
y(x - 1) = 3x + 2 ⇒ xy - y = 3x + 2 ⇒ xy - 3x = y + 2
x(y - 3) = y + 2 ⇒ x = (y + 2)/(y - 3)
∴ f⁻¹(x) = (x + 2)/(x - 3)

FUNCTION NOTATION
f: A → B (f maps A to B)
dom(f) = A, codom(f) = B
range(f) ⊆ B
INJECTIVE TEST
Horizontal Line Test: A function is injective if no horizontal line intersects its graph more than once
COMPOSITION PROPERTIES
(f ∘ g)⁻¹ = g⁻¹ ∘ f⁻¹
But f ∘ g ≠ g ∘ f in general
INVERSE CONDITIONS
A function has an inverse iff it is bijective
The graphs of f and f⁻¹ are symmetric about y = x
EVEN & ODD FUNCTIONS
Even: f(-x) = f(x)
Odd: f(-x) = -f(x)
Most functions are neither!
PERIODIC FUNCTIONS
f(x + T) = f(x) for smallest T > 0
sin(x) has period 2π
tan(x) has period π
PIECEWISE FUNCTIONS
Different definitions on different intervals
Important for JEE problems
Example: |x| = { x if x ≥ 0, -x if x < 0 }
DOMAIN RESTRICTIONS
Denominator ≠ 0
√(expression) requires expression ≥ 0
log(expression) requires expression > 0

Chapter Summary

Key Concepts

  • Relation is a subset of Cartesian product, function is a special relation
  • Important relation types: reflexive, symmetric, transitive, equivalence
  • Function types: injective, surjective, bijective, polynomial, exponential, etc.
  • Composition of functions is associative but not commutative
  • Inverse functions exist only for bijective functions
  • Domain restrictions are crucial for defining functions properly

Important Formulas

Number of relations from A to B = 2n(A)×n(B)
Number of functions from A to B = n(B)n(A)
Composition: (g ∘ f)(x) = g(f(x))
Inverse: f⁻¹(f(x)) = x and f(f⁻¹(y)) = y
Even: f(-x) = f(x), Odd: f(-x) = -f(x)
