Introduction to Trigonometry

Trigonometry is the branch of mathematics that deals with the relationships between the sides and angles of triangles, particularly right triangles.

Trigonometry: Derived from Greek words "trigonon" (triangle) and "metron" (measure), it's the study of relationships involving lengths and angles of triangles.

Historical Context

  • Developed by ancient Greek, Indian, and Arab mathematicians
  • Hipparchus (190-120 BCE) created the first trigonometric table
  • Used in astronomy, navigation, engineering, and physics

JEE Focus: Trigonometry accounts for about 10-15% of JEE Main questions and is fundamental for calculus and physics problems.

JEE Problem Example:

If sinθ + cosθ = √2cosθ, prove that cosθ - sinθ = √2sinθ.

Solution: Square both sides: (sinθ + cosθ)² = 2cos²θ ⇒ sin²θ + cos²θ + 2sinθcosθ = 2cos²θ
Using sin²θ + cos²θ = 1 ⇒ 1 + 2sinθcosθ = 2cos²θ
Rearrange: 2sinθcosθ = 2cos²θ - 1 ⇒ sin2θ = cos2θ ⇒ tan2θ = 1
Therefore, cosθ - sinθ = √2sinθ can be verified by substitution.

Trigonometric Functions

The six basic trigonometric functions relate angles to ratios of side lengths in right triangles.

Primary Trigonometric Functions

sinθ = Opposite/Hypotenuse
cosθ = Adjacent/Hypotenuse
tanθ = Opposite/Adjacent

Reciprocal Functions

cosecθ = 1/sinθ
secθ = 1/cosθ
cotθ = 1/tanθ

Signs in Different Quadrants

Quadrant Positive Functions Negative Functions
I (0°-90°) All None
II (90°-180°) sin, cosec cos, sec, tan, cot
III (180°-270°) tan, cot sin, cosec, cos, sec
IV (270°-360°) cos, sec sin, cosec, tan, cot

JEE Focus: Questions often involve finding values of trigonometric functions for non-standard angles using identities.

Trigonometric Identities

Identities are equations that hold true for all values of the variables.

Pythagorean Identities

sin²θ + cos²θ = 1
1 + tan²θ = sec²θ
1 + cot²θ = cosec²θ

Angle Sum and Difference

sin(A±B) = sinAcosB ± cosAsinB
cos(A±B) = cosAcosB ∓ sinAsinB
tan(A±B) = (tanA ± tanB)/(1 ∓ tanAtanB)

Double Angle Formulas

sin2θ = 2sinθcosθ
cos2θ = cos²θ - sin²θ = 2cos²θ - 1 = 1 - 2sin²θ
tan2θ = 2tanθ/(1 - tan²θ)

Triple Angle Formulas

sin3θ = 3sinθ - 4sin³θ
cos3θ = 4cos³θ - 3cosθ
tan3θ = (3tanθ - tan³θ)/(1 - 3tan²θ)
JEE Problem Example:

Prove that: (sinA + cosecA)² + (cosA + secA)² = 7 + tan²A + cot²A

Solution:
Expand LHS: sin²A + 2 + cosec²A + cos²A + 2 + sec²A
= (sin²A + cos²A) + 4 + (1 + cot²A) + (1 + tan²A)
= 1 + 4 + 1 + cot²A + 1 + tan²A = 7 + tan²A + cot²A = RHS

Trigonometric Equations

Equations involving trigonometric functions that are satisfied by specific values of the variables.

General Solutions

sinθ = sinα ⇒ θ = nπ + (-1)ⁿα
cosθ = cosα ⇒ θ = 2nπ ± α
tanθ = tanα ⇒ θ = nπ + α

Solving Strategies

  1. Express equation in terms of single trigonometric function
  2. Factorize when possible
  3. Use identities to simplify
  4. Consider domain restrictions
  5. Find general solution first, then specific solutions

JEE Focus: Trigonometric equations often appear in JEE with domain restrictions or combined with other concepts.

JEE Problem Example:

Solve: 2cos²θ + 3sinθ = 0

Solution:
Convert to sin: 2(1 - sin²θ) + 3sinθ = 0 ⇒ 2sin²θ - 3sinθ - 2 = 0
Let x = sinθ: 2x² - 3x - 2 = 0 ⇒ x = [3 ± √(9 + 16)]/4 ⇒ x = 2 or x = -½
sinθ = 2 has no solution, sinθ = -½ ⇒ θ = nπ - (-1)ⁿ(π/6)

Transformations

y = asin(bx + c) + d
• |a| = amplitude
• 2π/|b| = period
• -c/b = phase shift
• d = vertical shift
JEE Problem Example:

Find the period of the function f(x) = 3sin(5x - π/2) + 4

Solution:
Period = 2π/|b| = 2π/5
